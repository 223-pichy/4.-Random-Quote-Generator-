# Random Quote Generator - Генератор случайных цитат

## Информация об авторе
**Имя и фамилия:** Гайдай Иван Владимирович

## Описание программы

**Random Quote Generator** - это GUI-приложение для генерации случайных цитат с пацанским/жизненным вайбом. Приложение позволяет получать случайные цитаты, добавлять свои, фильтровать историю по автору и теме, а также сохранять историю в формате JSON.

### Возможности:
- ✅ Генерация случайной цитаты из библиотеки
- ✅ Отображение истории сгенерированных цитат (с датой и временем)
- ✅ Фильтрация истории по автору
- ✅ Фильтрация истории по теме
- ✅ Добавление собственных цитат в библиотеку
- ✅ Автоматическое сохранение истории в JSON-файл
- ✅ Автоматическая загрузка истории при запуске
- ✅ Очистка всей истории (с подтверждением)
- ✅ Валидация вводимых данных (проверка пустых полей)

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd quote-generator
Запустите приложение:

bash
python quote_generator.py
Примеры использования
Генерация случайной цитаты:
Нажмите кнопку "Сгенерировать цитату"

В верхней части окна появится случайная цитата с автором

Цитата автоматически добавится в историю справа/внизу

Пример сгенерированной цитаты:

text
📖 Жиза, братан...
— Пацанский мудрец
Добавление своей цитаты:
Заполните все поля формы "Добавить свою цитату":

Текст: "Круто быть добрым, но лучше быть сильным"

Автор: "Братан с района"

Тема: "Сила"

Нажмите кнопку "Добавить цитату"

Новая цитата добавляется в библиотеку и может выпасть при генерации

Фильтрация истории:
По автору: Выберите автора из выпадающего списка в разделе "Фильтрация истории" и нажмите "Применить"

По теме: Выберите тему из выпадающего списка и нажмите "Применить"

Можно комбинировать оба фильтра одновременно

Для сброса фильтров нажмите "Сбросить"

Управление историей:
История сохраняется автоматически при каждой генерации

При следующем запуске история загружается автоматически

Для очистки всей истории нажмите "Очистить историю" и подтвердите действие

Валидация данных
Приложение проверяет:

При добавлении цитаты все поля (Текст, Автор, Тема) должны быть заполнены

При пустых полях выводится сообщение об ошибке

При очистке истории требуется подтверждение действия

При отсутствии файла с историей при запуске создаётся пустая история
